# composition

Lessons, handouts, and browser tools for private composition lessons in composition, orchestration, and notation, by Inés Thiebaut at CSU East Bay. Materials are built as the lessons land, so this collection grows over time, and score study runs through all of it: reading a score closely to see how a passage is made.

The hub holds close-reading guides for specific scores and passages, orchestration lessons that work from range and transposition through balance, doubling, and color, notation lessons on setting music clearly on the page, and browser-based tools for analysis and teaching: form, texture, register and orchestration maps, and process. Each tool is self-contained and opens in the browser with nothing to install.

Live at https://csuebmusic.github.io/composition/
